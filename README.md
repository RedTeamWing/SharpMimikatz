## SharpReflectiveDll
Csharp 反射加载dll

> 这个技巧是在SharpGen这个工具里面学到的。

使用.net3.5，一般情况下支持win7以上。

## Usage

```
SharpReflectiveDll.exe args
```
args就按照mimikatz正常的参数来就好。

因为spawndll一直不行，用.net加载基本没啥bug,就这样了。
